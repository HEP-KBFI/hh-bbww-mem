\title{Application of the matrix element method to the analysis of $\PHiggs\PHiggs \to \Pbottom\APbottom\PW^{+}\PW^{-}$ in the dilepton decay channel}

\begin{abstract}
We present the development of a discriminant for the analysis of Higgs boson pair ($\PHiggs\PHiggs$) production in the decay channel 
$\PHiggs\PHiggs \to \Pbottom\APbottom \, \PW^{+}\PW^{-} \to \Pbottom\APbottom \, \ellPlusnu\ellMinusnu$,
which separates the $\PHiggs\PHiggs$ signal from the large irreducible background that arises from top quark pair production in this channel.
The discriminant is based on matrix element method.
\end{abstract}
